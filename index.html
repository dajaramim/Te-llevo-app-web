Sobre TeLlevoApp

TeLlevoApp es una aplicación móvil que permite a los estudiantes que tienen vehículo acercar a otros estudiantes a sus hogares si estos van hacía el mismo sector.

Esta aplicación utiliza la api Mapbox para visualizar el mapa y escoger el destino.
